Introdução ao JavaScript e Projeto Integrado com HTML

1. Introdução ao JavaScript

JavaScript é uma linguagem de programação usada principalmente para criar interatividade nas páginas web.

Ela permite que você adicione funcionalidades dinâmicas ao seu site, como responder a eventos de usuário, modificar conteúdo, validar formulários, e muito mais. O JavaScript roda no navegador do usuário, tornando a experiência de uso rápida e responsiva.

Neste guia, você aprenderá o básico do JavaScript e como integrá-lo a uma página HTML.

2. Como Integrar JavaScript ao HTML

Para usar JavaScript em uma página HTML, você pode inserir o código JavaScript dentro de uma tag <script> diretamente no arquivo HTML ou em um arquivo separado com extensão .js, que é referenciado no HTML.

Exemplo de código HTML com JavaScript integrado:

<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Projeto JavaScript</title>
</head>
<body>

    <h1>Olá, JavaScript!</h1>
    <button onclick="mostrarMensagem()">Clique em mim</button>

    <script>
    function mostrarMensagem() {
        alert('Você clicou no botão!');
    }
    </script>
</body>
</html>

3. Explicação do Código

Neste exemplo simples, usamos JavaScript para exibir uma mensagem de alerta quando o usuário clica no botão. A função mostrarMensagem() é definida dentro da tag <script>, e é chamada quando o botão é clicado, graças ao atributo onclick do botão.

  • <button onclick="mostrarMensagem()">: Adiciona um botão que chama a função JavaScript ao ser clicado.
  • function mostrarMensagem() { ... }: Define a função que exibe um alerta com uma mensagem.

4. Próximos Passos

Para aprender mais, você pode explorar outros eventos e interações com o JavaScript, como manipulação de elementos do DOM (Document Object Model) e criação de funcionalidades mais avançadas.

Com prática, você entenderá melhor como o JavaScript pode criar experiências interativas nas suas páginas web.
